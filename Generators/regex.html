Regex Flags

  • g: global match (find all matches)
  • i: ignore case
  • m: multiline (^ and $ match line start/end)
  • s: dot matches newline
  • u: unicode
  • y: sticky (matches at lastIndex)

Common Regex Tokens

  • \d: digit, \w: word, \s: whitespace
  • ^: start, $: end, .: any char, [ ]: char class
  • ( ): group, |: or, *: 0+, +: 1+, ?: 0/1
